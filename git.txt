Git commands:
—— git init 			» Creates git repository in current folder
—— git status			» Shows status of each file in repository
—— git add file.ext 		» Adds files to Staged Snapshot
 - git add --all		» Adds all not ignored files to Staged Snapshot
—— git commit 			» Moves files from Snapshot to History
 — git commit -m "Commit message"
—— git log 			» Shows log of each file in History
 — git log --oneline		» Shorter version
 — git log file.ext 		» Shows log of file in History
—— git push origin master	» Pushes changes to remote depository
—— git remote add origin URL	» Links remote depository
—— git tag -a v1.0 -m "Text"	» Tags current snapshot
—— git checkout id		» Jumps to different log by id, tag or master
—— git revert id		» Reverts snapshot of choosen id from master
—— git reset			» Unstages files in git status
 — git reset --hard		» Changes tracked files to match recent commit
—— git clean -f			» Removes all untracked files
—— git diff master		» Shows diff between working tree and master
 — git diff origin/master	» Shows diff between origin and master
